## Thorium Reader Web

Thorium Web is a free, open-source, web-based reader for EPUB, audiobooks, and digital comics. Built with Next.js and modern web technologies, it delivers a fast, responsive, and accessible reading experience directly in your browser. Deployed globally via Cloudflare Pages, it provides instant access to your digital publications without installation.

### Platform Information

- **Base Technology**: Next.js with React and TypeScript
- **Supported Formats**: EPUB, audiobooks, digital comics
- **Deployment**: Cloudflare Pages (globally distributed)
- **Version**: 1.1.5+
- **License**: Open Source (BSD-3-Clause)
- **Repository**: [github.com/edrlab/thorium-web](https://github.com/edrlab/thorium-web)

### Key Features

- **EPUB Support**: Full rendering and navigation of EPUB publications
- **Responsive Design**: Optimized for desktop and tablet browsers
- **Accessible Reading Experience**: Designed for readers with disabilities
- **Customizable Interface**: Themes, font sizes, line heights, word and letter spacing
- **React Aria Components**: Industry-standard accessibility patterns
- **Real-time Rendering**: Fast and responsive publication display
- **Embeddable**: Use as a standalone app or embed in your own projects
- **Comprehensive Customization**: Breakpoints, themes, icons, docking system configuration
