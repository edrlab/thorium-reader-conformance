# Accessibility of This Website {#accessibility-of-this-website}

This website is designed with accessibility as a core principle. We are committed to ensuring that all users, including those with disabilities, can navigate and access the information presented here.

## Accessibility Features {#accessibility-features}

### Navigation {#navigation}
- Clear and logical navigation structure
- Keyboard accessible menu navigation
- Skip links to main content
- Proper heading hierarchy

### Content {#content}
- Descriptive link text that makes sense out of context
- Alternative text for images
- Semantic HTML markup
- Proper color contrast ratios

### Technical Compliance {#technical-compliance}
- World Wide Web Consortium (W3C) Web Accessibility Guidelines (WCAG) 2.0 Level AA compliance
- Responsive design for all screen sizes
- Compatible with assistive technologies including screen readers
- Keyboard navigation support

### Data Tables {#data-tables}
- Properly structured with headers
- Interactive tables with sorting and filtering capabilities
- Accessible to assistive technologies

## Accessibility Statement for Thorium Reader {#accessibility-statement-for-thorium-reader}

This website contains conformance reports for Thorium Reader. For information about the accessibility of Thorium Reader itself, please visit the [accessibility reports](/documents/desktop/accessibility-report/) section.

## Report an Issue {#report-an-issue}

If you encounter any accessibility barriers while using this website, please let us know through our [contact page](/documents/contact/). We take accessibility seriously and will work to address any issues you report.
